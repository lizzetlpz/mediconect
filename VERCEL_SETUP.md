# Frontend MediConnect - Vercel Deployment

## Configuración para Vercel

### 1. Frontend (Angular)
- **Servicio:** Vercel (gratis)
- **Repositorio:** GitHub
- **Build Command:** `ng build --configuration production`
- **Output Directory:** `dist/nombre-proyecto`

### 2. Backend Connection
- **API Base URL:** `https://mediconect-production.up.railway.app/api`
- **Environment:** Production

### 3. Deploy Steps
1. Crear proyecto en Vercel
2. Conectar repositorio GitHub
3. Configurar environment variables
4. Deploy automático
